# 個人情報保護規定

**制定日:** 2025年7月
**最終改訂日:** 2025年12月7日
**適用範囲:** App Showcase サービス

---

## 第1条（目的）

本規定は、App Showcase（以下「当サービス」といいます）が取り扱う個人情報の保護について、個人情報の保護に関する法律（以下「個人情報保護法」といいます）および関連法令に基づき、適切な取り扱いを確保することを目的とします。

---

## 第2条（定義）

本規定における用語の定義は、以下の通りとします：

1. **個人情報**: 生存する個人に関する情報であって、当該情報に含まれる氏名、メールアドレス、その他の記述等により特定の個人を識別できるもの
2. **個人データ**: 個人情報データベース等を構成する個人情報
3. **保有個人データ**: 当社が、開示、内容の訂正、追加または削除、利用の停止、消去および第三者への提供の停止を行うことのできる権限を有する個人データ
4. **本人**: 個人情報によって識別される特定の個人

---

## 第3条（個人情報の収集）

### 3.1 収集する個人情報の種類

当サービスは、以下の個人情報を収集します：

#### アカウント情報
- メールアドレス（必須）
- 氏名またはユーザー名（必須）
- パスワード（暗号化して保存）
- アカウント作成日時

#### プロフィール情報
- アバター画像（任意）
- 自己紹介文（任意）
- プロフィール更新日時

#### サービス利用情報
- 商品購入履歴
- お気に入り登録履歴
- レビュー・コメント投稿内容
- お問い合わせ内容
- チャットボット会話履歴（セッション中のみ）

#### 技術情報
- IPアドレス
- ブラウザ種別・バージョン
- デバイス情報
- アクセスログ
- Cookie情報

#### 決済情報
- Stripe経由での決済情報（カード情報は保存しません）
- 購入日時・金額
- 取引ID

### 3.2 収集方法

個人情報は、以下の方法により収集します：

1. **直接入力**: ユーザー登録、プロフィール編集、お問い合わせフォーム等
2. **自動収集**: Cookie、アクセスログ、行動履歴等
3. **外部サービス**: Supabase Auth、Stripe、OpenAI API等の連携サービス
4. **推薦エンジン**: Gorse APIによる行動分析データ

---

## 第4条（個人情報の利用目的）

### 4.1 利用目的

収集した個人情報は、以下の目的でのみ利用します：

#### サービス提供
- ユーザー認証・アカウント管理
- 商品の購入・決済処理
- お問い合わせ対応
- カスタマーサポート
- AIチャットボットによる自動応答
- パーソナライズされた商品推薦

#### サービス改善・開発
- サービス品質の向上
- 新機能の開発
- ユーザー行動の分析
- マーケティング分析
- セキュリティ向上

#### 連絡・通知
- 重要なお知らせの通知
- セキュリティアラート
- サービス更新情報
- 購入確認・領収書送付

#### 法令遵守
- 法令に基づく開示請求への対応
- 不正行為の防止・検出
- 利用規約違反の調査

### 4.2 利用目的の変更

利用目的を変更する場合は、変更前の利用目的と相当の関連性を有すると合理的に認められる範囲内で行い、変更後の利用目的について本人に通知または公表します。

---

## 第5条（個人情報の安全管理）

### 5.1 安全管理措置

当サービスは、個人情報の漏えい、滅失または毀損の防止その他の個人情報の安全管理のために、以下の措置を講じます：

#### 技術的安全管理措置
- **暗号化**: パスワードのハッシュ化、通信のSSL/TLS暗号化
- **アクセス制御**: 管理画面への認証・認可機能
- **ファイアウォール**: 不正アクセスの防止
- **ログ監視**: セキュリティインシデントの早期検知
- **定期バックアップ**: データ保全とディザスタリカバリ

#### 組織的安全管理措置
- 個人情報保護規定の策定・運用
- 個人情報取扱責任者の設置
- 従業員への教育・研修
- 定期的な監査・見直し

#### 人的安全管理措置
- 秘密保持契約の締結
- 個人情報取扱者の限定
- 退職後の秘密保持義務

#### 物理的安全管理措置
- サーバーのアクセス制限（AWS、Vercel、Supabase）
- 機器・媒体の持ち出し制限
- データセンターの物理的セキュリティ

### 5.2 利用するクラウドサービス

当サービスは、以下のクラウドサービスを利用しています：

| サービス | 用途 | 提供元 |
|---------|------|--------|
| Supabase | データベース・認証 | Supabase Inc. |
| Vercel | ホスティング | Vercel Inc. |
| AWS | バックエンドサーバー | Amazon Web Services |
| Stripe | 決済処理 | Stripe, Inc. |
| OpenAI API | AIチャットボット | OpenAI |
| Gorse | 推薦エンジン | Gorse プロジェクト |

## 第6条（個人情報の第三者提供）

### 6.1 第三者提供の原則

当サービスは、以下の場合を除き、本人の同意なく個人情報を第三者に提供しません：

1. 法令に基づく場合
2. 人の生命、身体または財産の保護のために必要がある場合であって、本人の同意を得ることが困難であるとき
3. 公衆衛生の向上または児童の健全な育成の推進のために特に必要がある場合であって、本人の同意を得ることが困難であるとき
4. 国の機関もしくは地方公共団体またはその委託を受けた者が法令の定める事務を遂行することに対して協力する必要がある場合であって、本人の同意を得ることにより当該事務の遂行に支障を及ぼすおそれがあるとき

### 6.2 外部サービスとの連携

以下のサービスとは、サービス提供のために必要な範囲で個人情報を共有します：

- **Stripe**: 決済処理のため（カード情報は直接取り扱いません）
- **OpenAI API**: チャットボット機能のため（会話内容のみ、個人識別情報は除外）
- **Supabase**: データベース管理のため

---

## 第7条（個人情報の保管期間）

### 7.1 保管期間

個人情報は、利用目的を達成するために必要な期間に限り保管します：

| データ種別 | 保管期間 |
|-----------|---------|
| アカウント情報 | アカウント削除まで |
| 購入履歴 | 最終購入から7年間（商法要件） |
| お問い合わせ履歴 | 対応完了から3年間 |
| アクセスログ | 最終アクセスから6ヶ月間 |
| チャット履歴 | セッション終了時に削除 |

### 7.2 保管期間経過後の措置

保管期間が経過した個人情報は、速やかに消去または匿名化します。

---

## 第8条（本人の権利）

### 8.1 開示請求権

本人は、当社に対して、保有個人データの開示を請求することができます。

### 8.2 訂正・追加・削除請求権

本人は、保有個人データの内容が事実でないときは、訂正、追加または削除を請求することができます。

### 8.3 利用停止・消去請求権

本人は、以下の場合、保有個人データの利用停止または消去を請求することができます：

1. 利用目的の範囲を超えて取り扱われている場合
2. 不正の手段により取得された場合
3. 利用する必要がなくなった場合

### 8.4 第三者提供停止請求権

本人は、保有個人データが本人の同意なく第三者に提供されている場合、その停止を請求することができます。

### 8.5 請求方法

権利行使の請求は、以下の方法で受け付けます：

- **方法**: お問い合わせフォームまたはメール
- **本人確認**: 登録メールアドレスからの送信
- **回答期限**: 請求受理後2週間以内

---

## 第9条（Cookie等の利用）

### 9.1 Cookieの利用目的

当サービスは、以下の目的でCookieを使用します：

- ログイン状態の維持
- ユーザー設定の保存
- サービス利用状況の分析
- セキュリティ対策

### 9.2 Cookie の無効化

ユーザーは、ブラウザの設定によりCookieを無効化できますが、一部機能が利用できなくなる場合があります。

---

## 第10条（未成年者の個人情報）

### 10.1 保護者の同意

15歳未満の未成年者が当サービスを利用する場合、保護者の同意を得る必要があります。

### 10.2 特別な配慮

未成年者の個人情報については、特に慎重な取り扱いを行います。

## 第11条（個人情報保護管理体制）

### 11.1 個人情報保護責任者

当サービスは、個人情報保護責任者を設置し、個人情報の適切な管理を行います。

### 11.2 従業員の教育

個人情報を取り扱う従業員に対し、定期的な教育・研修を実施します。

### 11.3 監査

個人情報保護の実施状況について、定期的に内部監査を実施します。
