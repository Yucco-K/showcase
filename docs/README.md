# ドキュメント

このディレクトリには、プロジェクトの各種ドキュメントが含まれています。

## ファイル一覧

### 📋 [FAQ.md](./FAQ.md)

- アカウント関連の質問

- 技術サポート
- その他の質問

### 📜 [TERMS_OF_SERVICE.md](./TERMS_OF_SERVICE.md)

- サービスの利用条件

### 🔒 [PRIVACY_POLICY.md](./PRIVACY_POLICY.md)

- セキュリティ対策

## 使用方法

これらのドキュメントは、以下の方法で利用できます：

### 1. 直接参照

各 md ファイルを直接参照して内容を確認できます。

### 2. Web ページでの表示

Information ページ（`/information`）でこれらのドキュメントを表示できます。

### 3. リンクでの参照

他のページから以下のようにリンクできます：

```markdown
[FAQ を確認する](./FAQ.md)
[利用規約を読む](./TERMS_OF_SERVICE.md)
[プライバシーポリシーを確認する](./PRIVACY_POLICY.md)
```
